# Job Scraper Pro

A Python web scraping project that collects job listings from a website and saves them to a CSV file.

## Features

- Scrape job titles
- Scrape company names
- Scrape locations
- Save data to CSV

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python scraper.py
```

## Output

The program generates:

```text
jobs.csv
```
